# mini-server-with-golang

Mini servidor HTTP escrito em Go (Golang).

## Descrição

Este projeto é um servidor HTTP minimalista desenvolvido com a linguagem Go. O objetivo é demonstrar como criar e executar um servidor web simples sem o uso de frameworks externos, utilizando apenas a biblioteca padrão da linguagem.

## Tecnologias utilizadas

- Go (Golang)

## Requisitos

- Go 1.18 ou superior instalado na máquina

## Como executar

Clone o repositório:

```bash
git clone https://github.com/alexarlindo/mini-server-with-golang.git
cd mini-server-with-golang
```

Execute o servidor:

```bash
go run main.go
```

O servidor será iniciado e ficará aguardando requisições na porta configurada (por padrão, porta 8080).
